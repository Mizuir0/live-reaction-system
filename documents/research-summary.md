# 研究概要（詳細まとめ） 

1. 研究テーマ 

「観客の自然なリアクションを用いたオンラインライブ装飾システムの開発」 
オンラインライブ視聴中の観客の身体的リアクションを自動検出し、それをもとにライブ画面をリアルタイムで装飾することで、観客間の一体感と没入感を高める Web システムの開発を目的とする。 

2. 研究背景と課題意識 

* コロナ禍以降、オンラインライブ配信が急速に普及し、新たなエンターテインメント形態として定着 

* 対面ライブでは、観客の歓声・動き・拍手などが演出の一部となり、会場全体の「盛り上がり」を共有できる。 

* 一方、オンラインライブでは、観客が個別に視聴しているため他の観客の反応が見えず、観客同士の一体感や**「ノリ」の共有**が難しいという課題がある。 

* コメント機能は能動的すぎてライブへの集中を妨げる可能性があり、VR ライブは機材コストが高い。 
    →より自然で誰でも使える方法で「観客の存在感」を表現する仕組みが求められている。 

3. 研究目的 

オンラインライブにおいて、 
* 観客が自然に取る身体的リアクション（顔の表情、動き、拍手、歓声など）を自動で取得し、

* それらをもとに視聴画面上のエフェクトとして「全体の盛り上がり」や「雰囲気」を可視化することで、 

* コメントや機材に頼らず、自然な一体感を演出できる仕組みを実現する。 

* 目的 (Objective): これらの課題を解決するため、本研究では PC やスマートフォンの標準的なカメラ・マイクのみを使用し、視聴者の**「自然な」身体的リアクション（頭の動き、手拍子、歓声など）をリアルタイムで収集・集約する。そして、集約結果に応じた視覚的エフェクトをライブ画面の周囲に表示することで、視聴者の集中を妨げずに「他者の存在」や「全体の盛り上がり」を可視化し、オンラインライブにおける社会的臨場感と一体感**を向上させる Web アプリケーションを開発・評価する。 

4. システム構成 
提案システムの特徴： 

* PC やスマートフォンの標準カメラ・マイクのみで動作（追加機器不要） 

* 視聴者の無意識的・自然な振る舞いを活用（Zero-effort participation） 

* プライバシーに配慮した集約型表現（個人の映像は共有されない） 

アーキテクチャ： 

1.  フロントエンド  (クライアント側): 

    * 映像表示: 埋め込まれた YouTube Live プレーヤーでライブ映像を再生する。 

    * リアクション収集: PC・スマートフォンのカメラ・マイクにアクセスし、視聴者の身体動作（頭、手）、表情、音声（手拍子、歓声）をリアルタイムで解析する。（例：MediaPipe, Web Audio API） 

    * データ送信:  解析結果（「手拍子をした」「笑顔になった」等）を  WebSocket を通じてバックエンドサーバーに送信する。 

    * エフェクト描画:  バックエンドから受信した「エフェクト指示」に基づき、Canvas を用いてライブプレーヤーの周囲に指定されたエフェクトを描画する。 

2.  バックエンド (サーバー側): 

    * データ集約: WebSocket を介し、複数の視聴者からリアクションデータをリアルタイムで受信・集約する。 

    * エフェクト決定: 全体のリアクションデータを分析し、「全体の盛り上がり度」や「特定のリアクションの多さ」を判断し、表示すべきエフェクトの種類（例：「盛り上がり大」「手拍子多発」）を決定する。 

    * 指示送信: 決定した「エフェクト指示」を、全視聴者のフロントエンドに一斉にブロードキャストする。 

データフロー： 

1.  各視聴者端末でリアクション検出（30fps） 

2.  検出結果を WebSocket 経由でサーバーへ送信 

3.  サーバーで全視聴者のデータを集約・分析 

4.  盛り上がり度に応じたエフェクト指示を全端末へ配信 

5.  各端末でライブ画面周囲にエフェクトを描画 

5.  評価実験 

システムの有効性を検証するため、以下の3群比較実験を実施する。 

* 実験群  (Experimental Group):  本システムを利用する（リアクションに応じたエフェクトが表示される）。 

* 対照群1 (Placebo Group): 本システムを利用するが、ランダムなエフェクトが表示される（リアクションとは連動しない）。 

* 対照群2 (Control Group): システムを利用せず、YouTube のライブ動画をそのまま視聴する。 

評価指標: 

1.  主観評価: 視聴後にアンケートを実施し、「一体感」「楽しさ」「ライブへの熱中度」

「他者の存在感」などを測定する。 

* 臨場感尺度（Presence Questionnaire） 

* 視聴満足度（5段階リッカート尺度） 

* 社会的存在感（Social Presence Scale） 

2. 客観評価: システムが収集したリアクションデータ（頭の動きの量、手拍子の回数など）を群間で比較し、システムの利用が視聴者の行動をより活発にしたかを分析する。 

* リアクション頻度・持続時間 

* 視線滞留時間（エフェクト領域  vs  ライブ画面） 

* 認知負荷（NASA-TLX） 

6.  期待される成果と将来展望 

本研究は、**「低コスト（標準デバイス）」かつ「低負荷（自然な反応）」**な方法で、リモート環境における集団的な体験（一体感・社会的臨場感）を向上させるシステムデザインを提案・実装・評価するものである。この知見は、オンラインライブだけでなく、Web 会議、オンライン授業、e スポーツ観戦など、多様なリモートコミュニケーションの質的向上に寄与することが期待される。
